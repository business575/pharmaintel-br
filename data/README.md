# data/

Esta pasta contém todos os dados do projeto PharmaIntel BR.

## Subpastas

- `raw/` — Dados brutos baixados diretamente das APIs (Comex Stat, ANVISA, BPS, ComprasNet). **Não commitar.**
- `processed/` — Dados após limpeza e transformação pelo pipeline ETL.
- `exports/` — Relatórios e arquivos gerados para download pelo usuário na plataforma.
