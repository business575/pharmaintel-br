# src/

Código-fonte principal da plataforma PharmaIntel BR.

## Subpastas

- `integrations/` — Conectores para as APIs públicas (Comex Stat, ANVISA, BPS, ComprasNet)
- `agents/` — Lógica do agente de IA (Groq/Llama 3.3 70B) para análise estratégica
- `utils/` — Funções auxiliares reutilizáveis (formatação, logging, retry, encoding)
