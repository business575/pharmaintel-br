# dashboard/

Interface web do PharmaIntel BR construída com Streamlit.

## Subpastas

- `components/` — Componentes Streamlit reutilizáveis (filtros, gráficos, tabelas, cards de KPI)
- `assets/` — Arquivos estáticos: logo, imagens, CSS customizado

## Ponto de entrada

O arquivo principal do dashboard será `app.py` na raiz do projeto.
